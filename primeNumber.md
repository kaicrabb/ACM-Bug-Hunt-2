## Prime Number
[Code](primeNumber.py) <br>

Jack wants to know if some numbers he inputs are prime numbers or not. Jack has created a program, but is unable to get the correct prime numbers. Help him out!

**Input**<br>
The input consists of a positive int, N, which is the total numbers we will determine whether or not are prime. The N lines after will have a positive number (1 <= N <= 1000)

**Output**<br>
True/False based on the number

<br><br>
### Sample Output
**Input**
```
3
17
4
5
```
**Ouput**
```
True
False
True
```
